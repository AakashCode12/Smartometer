How to use OCR?

Step 1

Click on Choose file button in the website.

Step 2

After clicking on the Choose file button, choose the file from which you want to extract the text & click on the convert button.

Step 3

Our inbuilt engine on the server side will process your image using tesseract engine.

Step 4

Wait for few Seconds and the pdf file will be downloaded to your device from which you can extract you text.

What is OCR?

Optical character recognition or optical character reader (OCR) is the electronic or mechanical conversion of images of typed, handwritten or printed text into machine-encoded text, whether from a scanned document, a photo of a document, a scene-photo (for example the text on signs and billboards in a landscape photo) or from subtitle text superimposed on an image (for example: from a television broadcast).

Here for this project, we have used tesseract which is Google's open source engine for optical character recognition (OCR). I have made the back-end using Node.js using the following modules of node:

1) Tesseract Module
2) Express
3) FileSystem
4) Multer
5) Nodemon
